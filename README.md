# 🤝 Study Mate - Find Your Perfect Academic Partner

**Study Mate** is a modern, full-stack web application designed to help students connect with compatible study partners. Whether you're looking for someone to tackle complex calculus or practice a new language, Study Mate makes collaboration simple and efficient.

---

### 🌐 Project Links
- **Live Deployment:** [https://study-mate-22c7d.web.app/](https://study-mate-22c7d.web.app/)
- **Server API:** [https://study-mates-server-site.vercel.app](https://study-mates-server-site.vercel.app)

---

## ✨ Features

- **Secure Authentication:** Integrated with Firebase for Email/Password and Google One-Tap login.
- **Partner Matching:** Browse and search for partners based on subjects, experience levels, and study modes.
- **Personalized Connections:** Add, update, and manage your list of study partners in a dedicated dashboard.
- **Real-time Feedback:** Uses SweetAlert2 for beautiful, non-intrusive user notifications and confirmations.
- **Fully Responsive:** A seamless experience across mobile, tablet, and desktop devices using Tailwind CSS.
- **CRUD Operations:** Full ability to Create, Read, Update, and Delete connection requests and profiles.

---

## 🛠 Tech Stack

### Frontend
- **React.js:** Component-based UI development.
- **React Router:** For smooth, client-side navigation.
- **Tailwind CSS:** For modern, utility-first styling.
- **DaisyUI:** For high-quality, accessible UI components.
- **SweetAlert2:** For professional-grade user alerts.

### Backend & Database
- **Node.js & Express:** Scalable server-side logic and API routing.
- **MongoDB:** Flexible NoSQL database for storing user and connection data.
- **Firebase:** Reliable authentication and production-ready hosting.
- **Axios:** For clean and efficient API requests.
